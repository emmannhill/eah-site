What’s Included?
After purchase, you’ll receive a download link for a ZIP file containing the OTF font file and a PDF with installation instructions and licensing details.
One License = One User
If someone else wants to use the font, they’ll need their own license. Organizations with up to 50 employees may use a single license internally. Larger organizations must reach out for custom extended licensing.
You'll also need a custom extended license if:
  • Your website receives over 10,000 monthly page views
  • Your business earns over $1M annually
  • The font is used for film or TV titles, logos, or credits
  • The font appears in magazines or newspapers
  • The font is used in a national or international advertising campaign
  • The font is embedded in apps, software, eBooks, or mass-distributed digital files
Questions?
If you’re unsure which license you need or have any other questions, email hello@emilyannhill.com. These terms help protect the work that goes into creating each font while giving you the freedom to make something great. Thanks for supporting independent type design!
